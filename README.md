# java-lotto-precourse
구현 기능 목록
1. 로또 발매기 구현
- 1-1. 로또 구입 금액 입력받기
- 1-2. 로또 당첨 번호 입력받기
- 1-3. 로또 당첨 번호 파싱(쉼표를 기준으로 파싱)
- 1-3. 보너스 번호 입력받기

2. 로또 발행 기능
- 2-1. 발행할 로또 수량 계산
- 2-2. 로또 발행하기

3. 로또 당첨 결과 계산하기
- 3-1. 발행한 로또 티켓마다 등수 매기기
- 3-2. 당첨된 로또 티켓의 전체 상금 계산하기
- 3-3. 발행한 로또 티켓마다 로또 번호와 당첨 번호 비교하기
  
4. 당첨 내역 출력
- 4-1. 당첨 내역 출력하기

5. 수익률 계산
- 5-1. 소수점 둘째 자리에서 반올림

6. 등수 및 상금 관리 (Rank Enum)
- 6-1. 등수 정의
- 6-2. 등수 판별 메서드 정의
- 6-3. 상금 조회 기능
- 6-4. 일치 조건 판별
